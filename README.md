# OpenClaw Smart Weather Skill

智能天气助手 - 获取天气并提供穿衣、通勤建议的 OpenClaw 技能包。

## 功能特性

- 🌤️ 实时天气查询（支持全球城市）
- 👕 智能穿衣建议（根据温度和天气状况）
- 🚗 通勤出行建议（步行/骑行/驾车）
- 🏃 活动建议（户外/室内）
- 🌏 支持中文城市名称

## 安装

通过 OpenClaw 安装：

```bash
# 方式1：通过 skill 名称安装（如果已发布到 npm）
openclaw skills install openclaw-skill-smart-weather

# 方式2：本地安装（开发测试）
openclaw skills install /path/to/smart-weather
```

## 使用方法

安装后，AI 会自动识别以下场景并使用此技能：

- 询问天气："北京天气怎么样？"
- 询问预报："明天会下雨吗？"
- 穿衣建议："今天穿什么？"
- 通勤建议："适合骑车上班吗？"
- 活动建议："适合户外活动吗？"

## 使用示例

### 示例 1：查询天气

**用户：** "明天北京天气咋样，去旅游要穿什么？"

**AI 响应：**
```
🌤️ 北京天气预报（3月31日）

【明日预报】
☀️ 晴天为主，12-20°C
- 早上：晴天，12-13°C，微风
- 中午：晴天，17°C，微风
- 傍晚：晴天，20°C，西南风
- 夜间：零星小雨，17°C

👕 穿衣建议
- 基础搭配：长袖T恤 + 薄外套/卫衣 + 长裤
- 早上较冷（12°C）：可以加件薄毛衣或风衣
- 中午回暖（17-20°C）：可脱掉外套，穿长袖T恤
- 晚上有雨：建议带把折叠伞
- 鞋子：运动鞋或休闲鞋均可

🎒 旅游建议
✅ 非常适合旅游！
- 全天晴天，光线好，适合拍照
- 温度舒适，不冷不热
- 风力适中，体感舒适

📸 推荐景点（适合晴天游览）
- 故宫、天安门广场（开阔地带，拍照好看）
- 颐和园、天坛公园（户外散步）
- 八达岭长城（注意防晒）
```

## 技术细节

### 数据来源

使用 wttr.in API 获取天气数据：
- 无需 API 密钥
- 支持全球城市
- 提供 3 天预报

### 支持的查询格式

- 城市名：`成都`、`北京`、`上海`
- 国际城市：`Tokyo`、`New+York`、`London`
- 机场代码：`CTU`、`PEK`、`PVG`

### 依赖工具

- `curl`：用于请求 wttr.in API

## 穿衣建议逻辑

### 温度范围

| 温度范围 | 建议衣物 |
|---------|---------|
| ≤ 5°C | 羽绒服、厚棉衣、毛衣、保暖内衣、围巾、手套、帽子 |
| 5-10°C | 厚外套、羽绒服、毛衣、长裤、围巾 |
| 10-15°C | 薄外套、风衣、卫衣、长袖T恤、长裤 |
| 15-20°C | 长袖T恤、薄外套、长裤/牛仔裤 |
| 20-25°C | 短袖T恤、薄长裤 |
| 25-30°C | 短袖、短裤、薄衫 |
| ≥ 30°C | 轻薄透气衣物、防晒 |

### 天气状况

- **晴天**：注意防晒，准备太阳镜、防晒霜
- **多云/阴天**：可适当加一件薄外套
- **小雨**：带伞，穿防水鞋或靴子
- **中雨/大雨**：穿防水外套、雨靴、打伞
- **下雪**：穿羽绒服、防滑鞋、手套、帽子

## 注意事项

1. wttr.in 有频率限制，避免短时间内重复请求
2. 预报数据仅供参考，具体以官方气象台为准
3. 穿衣和通勤建议基于一般经验，需结合个人体质调整
